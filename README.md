## PetMatch

Java console application connected to a PostgreSQL database for managing pet adoptions, with a lifestyle-based compatibility quiz.

## Technologies
Java, PostgreSQL, JDBC, PreparedStatement

## Features
- Browse and adopt pets by species
- Normalized relational schema with 5 tables
- Secure CRUD operations using PreparedStatement
- Lifestyle-based compatibility quiz that recommends pet species

## How to Run
1. Set up PostgreSQL and run `Database.sql`
2. Update database credentials in the connection file
3. Run `Main.java`
